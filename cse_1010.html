CSE1010 - JAVASCRIPT

JavaScript, in of itself isn't the most useful (Still quite useful, just not the most) of programming languages. Although when used alongside HTML & CSS, it can make some really powerful websites. While HTML and CSS deal with what you see, JavaScript works behind the scenes and allows for many things you couldn't do with HTML & CSS. For example, with javascript, you can make a way to have accounts, or maybe a random number generator.

Javascript might seem a bit... complicated, but once you actually get to know more and more javascript, you will realize that it is actually quite simple! And very fun!
